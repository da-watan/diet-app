# Fit Note

スマホで毎日の体重・摂取カロリー・PFC・メモを記録する、アプリ風の静的Webツールです。

## 開き方

`index.html` をブラウザで開くだけで使えます。

```text
outputs/diet-app/index.html
```

サーバーやインストール作業は不要です。

## GitHub Pagesで公開する

### 新しいリポジトリで公開する場合

1. GitHubで新しいリポジトリを作成します。
2. この `outputs/diet-app` フォルダ内のファイルを、リポジトリのルートに配置します。
3. `index.html`、`styles.css`、`app.js`、`README.md`、`.nojekyll` をコミットして `main` ブランチへ push します。
4. GitHubのリポジトリ画面で `Settings` → `Pages` を開きます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. `Branch` を `main`、フォルダを `/ (root)` にして保存します。
7. 数分後に表示されるURLへアクセスします。

公開URLは通常、次の形式になります。

```text
https://ユーザー名.github.io/リポジトリ名/
```

### 既存リポジトリで公開する場合

既存リポジトリに置く場合も、GitHub Pagesの公開対象フォルダにこのアプリ一式を配置してください。公開対象を `/docs` にしたい場合は、ファイル一式を `docs/` 配下に入れて、Pages設定で `main` / `docs` を選びます。

## 主な機能

- 目標体重、期限、維持カロリーの設定
- 身長からBMIを表示
- 体重、摂取kcal、PFC、メモの記録
- 達成率、7日平均、直近変化、記録日数、残りkg、期限までの日数の表示
- 体重推移グラフ
- 履歴の確認と個別削除
- サンプルデータ追加
- JSON形式でのデータ書き出し
- JSON形式でのバックアップ復元
- GitHub Pages公開時のPWA対応

## スマホでアプリ風に使う

GitHub PagesなどのHTTPS環境で公開すると、ブラウザからホーム画面に追加できます。

- iPhone: Safariで公開URLを開き、共有ボタンから「ホーム画面に追加」
- Android: Chromeで公開URLを開き、メニューから「ホーム画面に追加」

ホーム画面から開くと、通常のWebページよりアプリに近い見た目で使えます。

## データ保存

記録データはブラウザの `localStorage` に保存されます。

同じブラウザ・同じ端末であれば、ページを閉じても記録は残ります。別の端末や別のブラウザには自動同期されません。

## ファイル構成

```text
outputs/diet-app/
  index.html   画面構造
  styles.css   スマホ向けレイアウトと見た目
  app.js       記録、集計、グラフ描画、保存処理
  manifest.webmanifest  PWA設定
  service-worker.js     オフライン再表示用キャッシュ
  icon.svg     ホーム画面用アイコン
  README.md    この説明
```

## 使い方

1. 「今日」タブで現在体重、目標体重、身長、期限、維持カロリーを入力します。
2. 「目標を保存」を押します。
3. 日々の体重、摂取kcal、PFC、メモを入力します。
4. 「進捗」タブで7日平均、BMI、残りkg、体重推移を確認します。
5. 「履歴」タブで過去の記録を確認・削除できます。
6. 右上の書き出しボタンでJSONバックアップを保存できます。
7. 「履歴」タブの「復元」から、書き出したJSONを読み込めます。

## 注意

このツールは健康管理の補助用です。体調不良、持病、妊娠中、摂食障害の不安がある場合は、医師や専門家に相談してください。
